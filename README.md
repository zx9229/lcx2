## Golang在windows下交叉编译linux程序

* 下载源码
```
go get -u -v github.com/zx9229/lcx2
```

* Golang在windows下交叉编译linux程序
```bat
set CGO_ENABLED=0
set GOOS=linux
set GOARCH=amd64
go build
```
